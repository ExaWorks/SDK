ExaWorks: Software Development Kit
==================================

.. image:: https://readthedocs.org/projects/exaworkssdk/badge/?version=latest
   :target: http://radicalanalytics.readthedocs.io/en/latest/?badge=latest
   :alt: Documentation Status

**ExaWorks Software Development Kit (SDK)** packages the Parsl, Flux, RADICAL-Pilot, and Swift/T workflow systems, offering a tested way to deploy them on high performance computing (HPC) platforms. Applications teams can draw from SDK's packages as needed, and develop and share workflows built on the various systems.

.. note::

   This project is under active development.
